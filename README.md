## Bulk Purchase App

### Pre-requisites

You need to install the following to make this work.

- `mongodb`
- `express`
- `react`
- `node`
- `mongoose`

## Running the app

Run Mongo daemon:

```
sudo mongod
```

Mongo will be running on port 27017.

Run Express:

```
cd backend/
npm install
nodemon server.js
```

Run React:

```
cd frontend
npm install/
npm start
```

Navigate to localhost:3000/ in your browser.
